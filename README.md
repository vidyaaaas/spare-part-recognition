# Spare Part Recognition

Python desktop application for recognizing spare parts from images or a webcam using OpenCV feature matching.

## Repository

```bash
git clone https://github.com/vidyaaaas/spare-part-recognition.git
cd spare-part-recognition
```

## Required Software

- Python 3.10 or newer
- Git
- Webcam, optional, for camera recognition
- Tkinter support for Python
  - On Windows, Tkinter is usually included with Python from python.org.

## Install Dependencies

Create and activate a virtual environment:

```bash
python -m venv venv
venv\Scripts\activate
```

Install Python dependencies:

```bash
pip install -r requirements.txt
```

## Environment Variables

This project does not require environment variables, API keys, passwords, or secret keys.

An `.env.example` file is included only as a placeholder for deployment or grading requirements.

## Database Setup

This project does not use a database.

Part metadata is stored in:

```text
parts_info.csv
```

Reference images are stored in:

```text
dataset/
```

## Run Locally

Start the desktop application:

```bash
python main.py
```

The application opens a GUI where you can:

- Select an image from your computer
- Use camera recognition with a webcam
- Reload the local dataset

## Test the Recognition System

Run the recognition test script:

```bash
python test_system.py
```

The test report is written to:

```text
recognition_test_report.csv
```

## Notes

- Keep `dataset/` and `parts_info.csv` in the project root.
- Webcam features require camera permission from the operating system.
- Temporary preview files generated by the GUI are ignored by Git.
